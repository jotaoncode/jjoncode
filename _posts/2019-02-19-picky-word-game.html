Picky Word Game

Objectives

To learn React native, google cloud, aws

iOS Words Picky
Google Play for android Words Picky

The art

Everything created for this game was done by scratch by me, and the very begining was kind of how should I start? Basically it is a book where you write synonyms / antonyms considering a hint of letters you can use for a given word.

From scratch

So everything started with some sort of idea, and no clue of how to create anything.

Not less, ugly but still a bit of a progress, I tried to do things in a different way. I started to search for a name...

After doing some brainstorming with words and so on... I came with the idea of creating a game of words where the person became a bit more "picky" in the chosing words he use when he wants to express.

So I tried to learn a bit of illustrator, and I feel that make a better advance.

Game Selection

The game currently has only one language, one country, but could support several countries and languages. The intention is to be able to choose what you want to learn in the language you decide.

Game Instructions

So what I want is that each person can get more words from synonyms and antonyms. Thats what the game is for, learning while playing.

Score

The results of the game is about how many points you did, and which where the answers of the game, this will help you to learn the game synonyms and antonyms eventhough you did not assert those values.

May be those results where completely different than the ones we expected to be, and then thats the point, to see how close are those words to be synonyms / antonyms in which situation.

Conclusion

In this case I learnt about react native. This project was done 2 years ago, but never published, and the new React Hooks provides a different approach on working with React, so this project got old. On the other hand I was mainly learning about back end things, nodejs, scala, AWS with ec2 and rds instances. This was deployed using a cloud function of google cloud.
