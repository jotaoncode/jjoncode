Forever learning

During the creation of this application I learnt about:

  • React Native: Navigation, Hooks, Async storage, etc...
  • D3 for react native
  • A bit, (just a bit) of Amplitude, as it is pretty a lot what can be done with this software. Similar to Google Tag Manager / Analytics etc.
  • A bit of Illustrator, for the asset creation.
  • A bit of video edition with Premiere Pro, for the short clip that promotes the application in youtube.
  • ... and well you know all those adventures that makes learning so cool :).
